    3 概率论

    条件概率

    因为已经知道事件A必然发生，所以只需在A发生的范围内考虑问题，即现在的样本空间为A。
    $$P(B \mid A)=\frac{n(A B)}{n(A)}$$
    因为在事件A发生的情况下事件B发生，等价于事件A和事件B同时发生，即AB发生
    $$P(B \mid A)=\frac{n(A B) / n(\Omega)}{n(A) / n(\Omega)}=\frac{P(A B)}{P(A)}$$
    P(B|A)与P(AB)
    相同点：事件A，B都发生了
    不同点：样本空间不同：在P(B|A)中，事件A成为样本空间；在P(AB)中，样本空间仍为$\Omega$ 。

    独立性
    设 A ， B 为两随机事件，
    若 $P(B \mid A)=P(B)$，即 $P(A B)=P(A) \times P(B)$
    即 $P(A \mid B)=P(A)$ 时，称 A ， B 相互独立。

    伯努利试验

    重复n次，每次具有独立性，考察每次出现事件的整体概率。
    如：股子连续抛掷n次，考察六点出现的次数及相应的概率：

    如果每次试验中事件 A 发生的概率为，
    $p(0<p<1)$

    则在 n 次贝努里试验中事件 A 恰好发生 k 次的概率为

    $P_{n}(k)=C_{n}^{k} p^{k} q^{n-k}, k=0,1, \cdots, n$

    其中 $q=1-p$，C代表了相同样本，不同顺序。

    古典概率指一个事件的概率
    条件概率指一个事件概率基础上的另一事件的概率
    伯努利指多个事件概率中共同事件的概率
    二维随机变量概率指概率矩阵，x为事件，Y为现象，由x,y构成平面的交叉点为概率。

    二维随机变量

    二维随机变量的联合函数
    F(x,y)表示随机点（X,Y）在以（x,y）为顶点且位于该点左下方无穷矩形内的概率。
    $$F(x, y)=P{(X \leq \boldsymbol{x}) \cap(Y \leq \boldsymbol{y})}$$

    联合分布函数F(x,y)表示矩形域概率
    $$\begin{aligned}
    & P\left(x_{1}<X \leq x_{2}, \quad y_{1}<Y \leq y_{2}\right) \
    =& F\left(x_{2}, y_{2}\right)-F\left(x_{2}, y_{1}\right)-F\left(x_{1}, y_{2}\right)+F\left(x_{1}, y_{1}\right)
    \end{aligned}$$

    二维连续型随机变量
    $$F(x, y)=\int_{-\infty}^{y} \int_{-\infty}^{x} f(u, v) d u d v$$

    边缘分布
    x方向求和为y的边缘分布
    y方向求和为x的边缘分布
    离散型
    $$\begin{array}{l}
    P\left{Y=y_{j}\right}=P\left{X<+\infty, \quad Y=y_{j}\right}=\sum_{i=1}^{\infty} p_{i j} \stackrel{\text { 记为 }}{=} p_{\bullet j} j=1,2, \cdots \
    P\left{X=x_{i}\right}=P\left{X=x_{i}, \quad Y<+\infty\right}=\sum_{j=1}^{\infty} p_{i j} \stackrel{\text { 记为 }}{==} p_{i \bullet} i=1,2, \cdots
    \end{array}$$
    连续型
    $$\begin{array}{l}
    f_{X}(x)=\int_{-\infty}^{+\infty} f(x, y) d y \
    f_{Y}(y)=\int_{-\infty}^{+\infty} f(x, y) d x
    \end{array}$$

    期望

    期望反映了随机变量的取值水平
    若每个事件概率乘以权重系数（次数）后求和收敛，则称其为随机变量X的数学期望。

    有点求事件平均次数概率的味道

    $$E(X)=\sum_{k=1}^{\infty} x_{k} p_{k}$$
    $$E(X)=\int_{-\infty}^{+\infty} x f(x) d x$$
    $$E(Z)=E(g(X, Y))=\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} g(x, y) f(x, y) d x d y$$
    属性

    1. 设 C 是常数，则有 E(C)=C
    2. 设 X 是一个随机变量， C 是常数，则有 E(C X)=C E(X)
    3. 设 X, Y 是两个随机变量，则有 E(X+Y)=E(X)+E(Y) 将上面三项合起来就是: E(a X+b Y+c)=a E(X)+b E(Y)+c
    4. 设 X, Y 是相互独立的随机变量，则有 E(X Y)=E(X) E(Y)

    方差
    期望的分散程度
    $$D(X)=E[X-E(X)]^{2}=E\left(X^{2}\right)-[E(X)]^{2}$$
    大数定理
    在试验不变的条件下，重复试验多次，随机事件的频率近似于它的概率。

    马尔可夫不等式
    $$P(X \geq a) \leq \frac{E(X)}{a} \quad X \geq 0, a>0$$
    切比雪夫不等式
    $$P{|X-E(X)| \geq \varepsilon} \leq \frac{\sigma^{2}}{\varepsilon^{2}}$$
    $$P{|X-E(X)|<\varepsilon} \geq 1-\frac{\sigma^{2}}{\varepsilon^{2}}$$

    中心极限定理
    不管总体是什么分布，任意一个总体的样本平均值都会围绕在总体的整体平均值周围，并且呈正态分布。

    熵
    物体内部的混乱程度。
    $$H(X)=-\sum_{x \in X} P(x) \log_{2} P(x)$$
    log=2.303ln

    似然

    根据概率（结果）推测样本（条件）
    离散型样本：（$p$ 已知 ）
    $$L(\theta)=\prod_{i=1}^{n} p\left(x_{i} ; \theta\right)$$
    连续型样本：
    $$L(\theta)=\prod_{i=1}^{n} f\left(x_{i} ; \theta\right)$$

    极大似然估计(找最大的)：
    $$L\left(x_{1}, x_{2}, \cdots, x_{n} ; \hat{\theta}\right)=\max {\theta \in \Theta} L\left(x{1}, x_{2}, \cdots, x_{n} ; \theta\right)$$

    补充：
    似然问题都是为了解决最大似然问题的，也就是在当前观测结果下，估计参数最有可能的值。
    $$\arg \max _{\theta} \mathrm{L}=\theta^{2}, \text { where } \theta \in[0,1]$$
    当 $\theta$=1时，L可以取到最大值。
